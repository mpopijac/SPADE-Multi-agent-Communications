Required packages:
	Python 2.7
	SPADE (Smart Python multi-Agent Development Environment) platform
	jsonpickle lib (https://jsonpickle.github.io/)


Starting:
	1. Run SPADE:
		configure.py localhost
		./runspade.py
		
	2. Run agents(Each agent runs in its terminal.):
		./Supplier.py
		./Repository.py
		./Sale.py
		./Sale2.py

Configuration file for repository: config-repository.txt
